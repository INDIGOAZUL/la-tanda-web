📖 Documentación para Desarrolladores

Bienvenido a la API de La Tanda. Esta guía te ayudará a integrar tus aplicaciones con nuestra plataforma.

🚀 Getting Started

La Tanda es una plataforma DeFi de ahorros cooperativos. Nuestra API REST te permite:

  • Gestionar grupos/tandas de ahorro
  • Operaciones de wallet (enviar, recibir, saldo)
  • Feed social y marketplace
  • Sistema de lotería integradas

🔐 Autenticación

La API usa JWT tokens. Flujo:

POST /api/auth/login 
{
  "email": "usuario@email.com",
  "password": "tu-password"
}

Response:

{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "user": { "id": "...", "email": "..." }
}

Usa el token en todas las requests:

Authorization: Bearer tu-token-aqui

📡 Endpoints Principales

� wallet

GET /api/wallet/balance - Obtener saldo
POST /api/wallet/transfer - Transferir fondos

👥 Grupos/Tandas

GET /api/groups - Listar grupos
POST /api/groups - Crear grupo

GET /api/feed - Obtener feed

💰 Ejemplos de Código

JavaScript/Fetch

// Login
const login = async () => {
  const res = await fetch('/api/auth/login', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ email: 'user@test.com', password: 'pass' })
  });
  const { token } = await res.json();
  return token;
};

// Obtener grupos
const getGroups = async (token) => {
  const res = await fetch('/api/groups', {
    headers: { 'Authorization': \`Bearer \${token}\` }
  });
  return await res.json();
};

cURL

# Login
curl -X POST https://la-tanda.com/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"user@test.com","password":"pass"}'

# Listar grupos
curl https://la-tanda.com/api/groups \
  -H "Authorization: Bearer tu-token"
