La Tanda Chain API

Getting Started

La Tanda es una plataforma fintech completa que ofrece tandas de ahorro rotativo, marketplace de servicios y productos, sistema de lotería predictiva, y más. Esta API permite a los desarrolladores integrar todas estas funcionalidades en sus aplicaciones.

Base URL

https://latanda.online

Características Principales

  • Tandas de Ahorro: Grupos de ahorro rotativo con distribuciones automáticas
  • Marketplace: Servicios profesionales, productos físicos y digitales
  • Lotería Predictiva: Predicciones ML para lotería de Honduras
  • Social Feed: Red social con posts, comentarios, likes y follows
  • Wallet Digital: Gestión de balance, depósitos y retiros
  • Blockchain: Integración con Polygon y futura La Tanda Chain

Primeros Pasos

  1. Crear una cuenta en latanda.online
  2. Obtener un JWT token mediante el endpoint de login
  3. Incluir el token en el header Authorization: Bearer <token>
  4. Explorar los endpoints disponibles en /docs
Cuenta Demo: Puedes usar demo@latanda.online / LaTandaDemo2026! para pruebas.

Authentication

La Tanda utiliza autenticación basada en JWT (JSON Web Tokens). Los tokens son válidos hasta que el usuario cierra sesión o cambia su contraseña.

Flujo de Autenticación

  1. Login: Enviar credenciales a POST /api/auth/login
  2. Recibir Token: El servidor responde con un JWT token
  3. Usar Token: Incluir en header Authorization: Bearer <token>
  4. Logout: Revocar token con POST /api/auth/logout

Ejemplo de Login

// Login request
const response = await fetch('https://latanda.online/api/auth/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    email: 'usuario@example.com',
    password: 'tu_contraseña'
  })
});

const data = await response.json();
const token = data.token;

// Usar el token en requests subsecuentes
const protectedResponse = await fetch('https://latanda.online/api/wallet/balance', {
  headers: {
    'Authorization': `Bearer ${token}`
  }
});

Revocación de Tokens

Los tokens son automáticamente revocados cuando:

  • El usuario cierra sesión (POST /api/auth/logout)
  • El usuario cambia su contraseña
  • Un administrador desactiva la cuenta
Seguridad: Nunca compartas tu token JWT. Guárdalo de forma segura y no lo incluyas en repositorios públicos.

Code Examples

Ejemplos prácticos de operaciones comunes con la API de La Tanda.

Obtener Balance de Wallet

const token = 'tu_jwt_token_aqui';

const response = await fetch('https://latanda.online/api/wallet/balance', {
  headers: {
    'Authorization': `Bearer ${token}`
  }
});

const data = await response.json();
console.log('Balance:', data.balance);
// Output: { success: true, balance: 1500.00, currency: 'HNL' }

Crear un Grupo de Ahorro

const createGroup = async (token) => {
  const response = await fetch('https://latanda.online/api/groups', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: 'Ahorro Familiar',
      description: 'Grupo de ahorro para vacaciones',
      contribution_amount: 500,
      frequency: 'weekly',
      max_members: 10
    })
  });

  const data = await response.json();
  return data.group_id;
};

Obtener Feed Social con Paginación

const getFeed = async (token, page = 1, limit = 20) => {
  const url = new URL('https://latanda.online/api/feed/social');
  url.searchParams.append('page', page);
  url.searchParams.append('limit', limit);

  const response = await fetch(url, {
    headers: {
      'Authorization': `Bearer ${token}`
    }
  });

  const data = await response.json();
  return data.events;
};

// Uso
const events = await getFeed(token, 1, 20);
events.forEach(event => {
  console.log(`${event.actor_name}: ${event.event_type}`);
});

Buscar Productos en Marketplace

const searchProducts = async (query, category = null) => {
  const url = new URL('https://latanda.online/api/marketplace/products');
  url.searchParams.append('search', query);
  if (category) url.searchParams.append('category', category);

  const response = await fetch(url);
  const data = await response.json();
  return data.products;
};

// Buscar laptops en categoría electrónica
const laptops = await searchProducts('laptop', 'electronics');

Rate Limits

La Tanda implementa límites de tasa para proteger la API de abuso y garantizar disponibilidad para todos los usuarios.

Límites por Categoría

Categoría Límite Ventana
Login/Auth 5 requests Por minuto
API General 5 requests Por segundo
Endpoints Públicos 10 requests Por segundo
Wallet Operations 3 requests Por segundo

Headers de Rate Limit

Cada respuesta incluye headers informativos sobre el estado del rate limit:

X-RateLimit-Limit: 5
X-RateLimit-Remaining: 3
X-RateLimit-Reset: 1709345678

Respuesta cuando se excede el límite

{
  "success": false,
  "error": "Rate limit exceeded",
  "message": "Too many requests. Please try again in 60 seconds.",
  "retry_after": 60
}
Importante: Si excedes los límites repetidamente, tu cuenta puede ser temporalmente suspendida. Implementa backoff exponencial en tu código.

Error Handling

La API utiliza códigos de estado HTTP estándar y devuelve errores en formato JSON consistente.

Formato de Respuesta de Error

{
  "success": false,
  "error": "Error type",
  "message": "Human-readable error message",
  "details": {
    "field": "Additional context"
  }
}

Códigos de Estado HTTP

Código Significado Descripción
200 OK Solicitud exitosa
201 Created Recurso creado exitosamente
400 Bad Request Parámetros inválidos o faltantes
401 Unauthorized Token JWT inválido o expirado
403 Forbidden No tienes permisos para esta acción
404 Not Found Recurso no encontrado
429 Too Many Requests Rate limit excedido
500 Internal Server Error Error del servidor

Errores Comunes

Token Revocado

{
  "success": false,
  "error": "Token revoked",
  "message": "Token ha sido revocado"
}

Balance Insuficiente

{
  "success": false,
  "error": "Insufficient balance",
  "message": "Balance insuficiente para esta operación",
  "details": {
    "required": 500,
    "available": 250
  }
}

Validación de Campos

{
  "success": false,
  "error": "Validation error",
  "message": "Datos inválidos",
  "details": {
    "email": "Email inválido",
    "password": "Contraseña debe tener al menos 8 caracteres"
  }
}

Manejo de Errores en Código

const makeRequest = async (url, options) => {
  try {
    const response = await fetch(url, options);
    const data = await response.json();

    if (!response.ok) {
      // Manejar errores HTTP
      if (response.status === 401) {
        // Token expirado, redirigir a login
        window.location.href = '/login';
      } else if (response.status === 429) {
        // Rate limit, esperar y reintentar
        const retryAfter = data.retry_after || 60;
        await new Promise(resolve => setTimeout(resolve, retryAfter * 1000));
        return makeRequest(url, options);
      } else {
        // Otros errores
        throw new Error(data.message || 'Error desconocido');
      }
    }

    return data;
  } catch (error) {
    console.error('Request failed:', error);
    throw error;
  }
};

La Tanda Chain API

La Tanda Chain es la blockchain propia de La Tanda, diseñada para transacciones rápidas y de bajo costo. Actualmente en desarrollo, con lanzamiento planificado en fases.

Estrategia Blockchain

La Tanda implementa una estrategia de dos fases:

  • Fase 1: Polygon PoS Mainnet (actual) - Integración con red establecida
  • Fase 2: La Tanda Chain propia - Red de 21 validadores con distribución geográfica
Documentación Completa: Lee la Estrategia Blockchain para detalles técnicos y económicos.

Integración con Web3

// Conectar a La Tanda Chain con ethers.js
import { ethers } from 'ethers';

const provider = new ethers.JsonRpcProvider('https://rpc.latandachain.com');

// Obtener balance
const balance = await provider.getBalance('0x...');
console.log('Balance:', ethers.formatEther(balance), 'LTD');

// Enviar transacción
const wallet = new ethers.Wallet(privateKey, provider);
const tx = await wallet.sendTransaction({
  to: '0x...',
  value: ethers.parseEther('10.0')
});

await tx.wait();
console.log('Transaction hash:', tx.hash);
