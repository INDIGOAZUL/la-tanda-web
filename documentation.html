🚀 Getting Started #

La Tanda es una plataforma de ahorro cooperativo digital que lleva las tandas tradicionales (ROSCAs) a la era digital para comunidades en Honduras.

Una tanda (ROSCA — Rotating Savings and Credit Association) es un sistema de ahorro comunitario tradicional en Latinoamérica. Un grupo de personas acuerda contribuir una cantidad fija de dinero periódicamente. En cada ronda, un miembro recibe el fondo completo. Es banca sin banco — confianza comunitaria como infraestructura financiera.

🔐 Authentication #

La Tanda utiliza JWT (JSON Web Tokens) para autenticación de API, con soporte para rotación de refresh tokens y blacklist.

Flujo de Login

1. Login del Usuario

POST /api/auth/login
Content-Type: application/json

{
  "email": "usuario@example.com",
  "password": "securepassword123"
}

# Respuesta
{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expires_in": 3600
}

2. Usar Token en Requests

GET /api/wallet/balance
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

3. Refresh Token

POST /api/auth/refresh
Content-Type: application/json

{
  "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}

Características de Seguridad

  • ✅ bcrypt 12 rounds - Hash de contraseñas
  • ✅ crypto.timingSafeEqual - Comparaciones sensibles
  • ✅ Refresh token rotación + blacklist (Redis)
  • ✅ WebSocket JWT auth - Autenticación en tiempo real
  • ✅ 2FA - Autenticación de dos factores

💻 Code Examples #

Login y Obtener Balance

JavaScript (fetch)

async function loginAndGetBalance(email, password) {
  // 1. Login
  const loginRes = await fetch('https://latanda.online/api/auth/login', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ email, password })
  });
  
  const { access_token } = await loginRes.json();
  
  // 2. Obtener balance
  const balanceRes = await fetch('https://latanda.online/api/wallet/balance', {
    headers: { 'Authorization': `Bearer ${access_token}` }
  });
  
  return await balanceRes.json();
}

cURL

# Login
curl -X POST https://latanda.online/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","password":"password123"}'

# Obtener balance
curl -X GET https://latanda.online/api/wallet/balance \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

Crear Grupo

const createGroup = async (token, groupName, monthlyAmount) => {
  const res = await fetch('https://latanda.online/api/groups', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${token}`
    },
    body: JSON.stringify({
      name: groupName,
      monthly_amount: monthlyAmount,
      max_members: 10
    })
  });
  return await res.json();
};

Obtener Social Feed

const getFeed = async (token, page = 1) => {
  const res = await fetch(`https://latanda.online/api/feed/social?page=${page}`, {
    headers: { 'Authorization': `Bearer ${token}` }
  });
  return await res.json();
};

⚡ Rate Limits #

Límites por Zona

Zona Límite Descripción
Auth 5/min Login, registro, endpoints de autenticación
API 5/s Endpoints API normales
General 10/s Requests generales

Respuesta Rate Limit

// HTTP 429 Too Many Requests
{
  "error": "Rate limit exceeded",
  "retry_after": 60
}

Mejores Prácticas

  • ✅ Implementar exponential backoff para retries
  • ✅ Cache de datos frecuentemente accedidos
  • ✅ Usar WebSocket para actualizaciones en tiempo real (no sujeto a rate limits HTTP)

❌ Error Handling #

Formato Estándar de Error

{
  "error": "ERROR_CODE",
  "message": "Descripción legible del error",
  "details": {} // Opcional: detalles adicionales
}

Códigos de Error Comunes

HTTP Código Descripción
400 INVALID_INPUT Parámetros de entrada inválidos
401 UNAUTHORIZED JWT faltante o inválido
403 FORBIDDEN Permisos insuficientes
404 NOT_FOUND Recurso no existe
409 CONFLICT Conflicto (ej: email duplicado)
429 RATE_LIMITED Límite de rate excedido
500 INTERNAL_ERROR Error interno del servidor

Ejemplo de Manejo de Errores

async function safeApiCall(url, token) {
  try {
    const res = await fetch(url, {
      headers: { 'Authorization': `Bearer ${token}` }
    });
    
    if (!res.ok) {
      const error = await res.json();
      throw new Error(`${res.status}: ${error.message}`);
    }
    
    return await res.json();
  } catch (err) {
    console.error('API Error:', err.message);
    // Manejar error...
  }
}

Consultar Balance LTD

ethers.js

import { ethers } from 'ethers';

const provider = new ethers.JsonRpcProvider(
  'https://rpc-amoy.polygon.technology/'
);

const contract = new ethers.Contract(
  '0x8633212865B90FC0E44F1c41Fe97a3d2907d9cFc',
  ['function balanceOf(address) view returns (uint256)'],
  provider
);

const balance = await contract.balanceOf('YOUR_WALLET_ADDRESS');
console.log('LTD Balance:', ethers.formatEther(balance));
