Getting Started

La Tanda is a digital cooperative savings platform (ROSCA system) that digitizes traditional rotating savings groups.

Groups & Tandas

Create, manage, and join savings groups with automated turn rotation.

Digital Wallet

Balance in Lempiras (HNL), contributions, withdrawals, transaction history.

Marketplace

Buy and sell products within the community.

Social Feed

Posts, media, polls, mentions, hashtags.

Base URL

Production: https://latanda.online/api
Development: http://localhost:3002/api

Authentication

La Tanda uses JWT (JSON Web Tokens) for authentication.

Important: Include token as: Authorization: Bearer <token>

Register

// Register
const response = await fetch('https://latanda.online/api/auth/register', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    name: 'John Doe',
    email: 'john@example.com',
    password: 'your-password',
    phone: '+50412345678'
  })
});
const data = await response.json();
// { success: true, token: "eyJ...", user: {...} }

Login

// Login
const response = await fetch('https://latanda.online/api/auth/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ email: 'john@example.com', password: 'your-password' })
});
const { token } = await response.json();

Code Examples

Get User Profile

const response = await fetch('https://latanda.online/api/users/me', {
  headers: { 'Authorization': `Bearer ${token}` }
});
const user = await response.json();

Create a Group

const response = await fetch('https://latanda.online/api/groups', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'My Savings Group',
    contributionAmount: 1000,
    frequency: 'weekly',
    maxMembers: 10
  })
});

cURL

# Login
curl -X POST https://latanda.online/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"john@example.com","password":"pass123"}'

# Get Profile
curl https://latanda.online/api/users/me \
  -H "Authorization: Bearer YOUR_TOKEN"
