Documentacion de La Tanda API

Inicio Rapido

Base URL: https://latanda.online

Autenticacion: Authorization: Bearer <token>

Formato: application/json

Ver Swagger UI interactivo

Endpoints Principales

  • POST /api/auth/login — Iniciar sesion
  • GET /api/feed/social — Feed social
  • GET /api/feed/social/following — Feed de seguidos
  • POST /api/feed/social/:id/like — Like a publicacion
  • POST /api/feed/social/:id/bookmark — Guardar publicacion
  • POST /api/feed/social/follow/:userId — Seguir usuario
  • POST /api/feed/social/report — Reportar contenido
  • GET /api/groups — Listar grupos
  • GET /api/contributions — Contribuciones

Autenticacion

1. Obtener token via POST /api/auth/login

2. Incluir en headers: Authorization: Bearer {token}

3. Tokens expiran en 7 dias

Codigos de Respuesta

  • 200 — Exito
  • 400 — Datos invalidos
  • 401 — No autenticado
  • 403 — Prohibido
  • 404 — No encontrado
  • 429 — Rate limit
  • 500 — Error del servidor
