Documentación para Desarrolladores

Bienvenido a la documentación oficial de la API de La Tanda Web3. Aquí encontrarás todo lo que necesitas para integrar tus aplicaciones con nuestra plataforma, desde la autenticación hasta las operaciones de cadena de bloques.

Empezando

La API de La Tanda es una interfaz RESTful que te permite interactuar con todos los servicios de la plataforma, incluyendo autenticación de usuarios, gestión de grupos (tandas), operaciones de billetera, feed social, mercado y más.

Para empezar, necesitarás:

  • Una cuenta activa en La Tanda Web3
  • Un token JWT de autenticación (obtenido después del inicio de sesión)
  • Conocimientos básicos de solicitudes HTTP y formato JSON

La URL base para todas las solicitudes a la API es: https://latanda.online/api/v1

Nota
Todas las solicitudes deben realizarse a través de HTTPS. Las solicitudes HTTP serán redirigidas automáticamente.

Autenticación

La API utiliza tokens JWT (JSON Web Tokens) para autenticar las solicitudes. Después de iniciar sesión exitosamente, recibirás un token de acceso que deberás incluir en todas las solicitudes subsiguientes.

Para autenticar una solicitud, incluye el token en el encabezado Authorization con el prefijo Bearer:

Encabezado de Autenticación 
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

Flujo de autenticación:

  1. Envía una solicitud POST a /auth/login con tu correo electrónico y contraseña
  2. Recibe un token de acceso y un token de actualización (refresh token)
  3. Incluye el token de acceso en todas las solicitudes a endpoints protegidos
  4. Cuando el token expire, usa el refresh token para obtener un nuevo token de acceso
Seguridad
Nunca compartas tu token JWT con terceros. Los tokens tienen una validez de 24 horas. Asegúrate de almacenarlos de forma segura en tu aplicación.

Vista General de API

La API se organiza en las siguientes categorías de endpoints:

Categoría Descripción Base Path
Auth Gestión de autenticación, registro, inicio de sesión y recuperación de contraseña /auth/*
Wallet Operaciones de billetera, consulta de saldo, transferencias de tokens LTD /wallet/*
Groups/Tandas Gestión de grupos, creación, unión, administración de miembros y ciclos de pago /groups/*
Social Feed Publicaciones, me gusta, comentarios y marcadores del feed social /feed/*
Marketplace Operaciones de compra y venta de bienes y servicios entre usuarios /marketplace/*
Lottery Participación en sorteos y consulta de resultados /lottery/*
Admin Funciones de administración de plataforma (solo para usuarios con permisos de administrador) /admin/*

Ejemplos de Código

A continuación, ejemplos de las operaciones más comunes usando cURL y JavaScript Fetch API:

Inicio de Sesión

cURL 
curl -X POST https://latanda.online/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "tu@correo.com",
    "password": "tu_contraseña"
  }'
JavaScript (Fetch) 
fetch('https://latanda.online/api/v1/auth/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    email: 'tu@correo.com',
    password: 'tu_contraseña'
  })
})
.then(response => response.json())
.then(data => {
  const accessToken = data.access_token;
  // Almacena el token de forma segura
})
.catch(error => console.error('Error:', error));

Obtener Grupos del Usuario

cURL 
curl https://latanda.online/api/v1/groups \
  -H "Authorization: Bearer TU_TOKEN_JWT"

Crear una Publicación en el Feed

JavaScript (Fetch) 
fetch('https://latanda.online/api/v1/feed/posts', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer TU_TOKEN_JWT'
  },
  body: JSON.stringify({
    content: 'Hola comunidad de La Tanda! 🎉',
    visibility: 'public'
  })
})
.then(response => response.json())
.then(data => console.log('Publicación creada:', data));

Límites de Tasa

Para garantizar la estabilidad de la plataforma, aplicamos los siguientes límites de tasa a las solicitudes API:

Endpoint Límite
Inicio de sesión /auth/login 5 solicitudes por minuto
Endpoints de API general 5 solicitudes por segundo
Recursos generales (assets, páginas) 10 solicitudes por segundo

Los encabezados de respuesta incluirán información sobre tu límite de tasa actual:

  • X-RateLimit-Limit: Número máximo de solicitudes permitidas por ventana
  • X-RateLimit-Remaining: Número de solicitudes restantes en la ventana actual
  • X-RateLimit-Reset: Timestamp UNIX cuando se reiniciará el límite
Límite Excedido
Si excedes el límite de tasa, recibirás una respuesta con código de estado 429 Too Many Requests. Espera el tiempo indicado en el encabezado Retry-After antes de realizar nuevas solicitudes.

Manejo de Errores

La API usa códigos de estado HTTP estándar para indicar el resultado de las solicitudes:

Código de Estado Descripción
200 OK Solicitud exitosa
201 Created Recurso creado exitosamente
400 Bad Request Solicitud inválida (parámetros faltantes o incorrectos)
401 Unauthorized Autenticación fallida o token inválido/expirado
403 Forbidden No tienes permisos para acceder al recurso
404 Not Found El recurso solicitado no existe
422 Unprocessable Entity Error de validación de datos
429 Too Many Requests Has excedido el límite de tasa
500 Internal Server Error Error interno del servidor

Todas las respuestas de error siguen un formato JSON estándar:

Formato de Respuesta de Error 
{
  "error": {
    "code": "INVALID_CREDENTIALS",
    "message": "Correo electrónico o contraseña incorrectos",
    "details": {}
  }
}

API de La Tanda Chain

La Tanda Chain es nuestra cadena de bloques nativa, construida con Cosmos SDK. Ofrecemos endpoints de API REST y RPC para interactuar directamente con la cadena:

  • API REST: https://latanda.online/chain/api/ (puerto 1317)
  • RPC: https://latanda.online/chain/rpc/ (puerto 26657)
  • Chain ID: latanda-testnet-1
  • Denominación de Token: ultd (1 LTD = 1,000,000 ultd)

Ejemplos de operaciones comunes en la cadena:

Consultar Saldo de una Dirección

cURL 
curl https://latanda.online/chain/api/cosmos/bank/v1beta1/balances/ltd1direccion...

Obtener Detalles de una Transacción

cURL 
curl https://latanda.online/chain/api/cosmos/tx/v1beta1/txs/HASH_DE_TRANSACCION

Listar Validadores Activos

cURL 
curl https://latanda.online/chain/api/cosmos/staking/v1beta1/validators?status=BOND_STATUS_BONDED
