Crear Propuesta via CLI

Las propuestas de gobernanza se crean directamente en La Tanda Chain via la linea de comandos. Necesitas tener un nodo configurado o acceso a un nodo RPC.

Requisitos

  • Tener latandad instalado (guia de instalacion)
  • Tener una wallet con fondos para el deposito minimo
  • Deposito minimo para propuesta: cargando...

Comando

# 1. Crear archivo de propuesta cat > propuesta.json << 'EOF' { "title": "Tu titulo aqui", "summary": "Descripcion detallada de la propuesta", "messages": [], "deposit": "10000000ultd", "metadata": "" } EOF # 2. Enviar la propuesta a la chain latandad tx gov submit-proposal propuesta.json \ --from tu-wallet \ --keyring-backend test \ --chain-id latanda-testnet-1 \ --fees 1000ultd \ --gas 300000 -y

Tipos de Propuesta

  • Cambio de Parametros — Modificar parametros del protocolo (staking, slashing, governance)
  • Software Upgrade — Proponer una actualizacion del software del nodo
  • Community Pool Spend — Solicitar fondos del community pool
  • Text Proposal — Propuesta de senal (sin ejecucion automatica)
