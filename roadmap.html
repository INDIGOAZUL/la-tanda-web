La Tanda Platform Roadmap

Building the World's Most Accessible Web3 Financial Platform

Last Updated: January 10, 2026 | Version: 3.27.1

Our Vision
Build the world's most accessible Web3 financial platform, combining traditional tandas (rotating savings and credit associations) with blockchain technology to serve the unbanked and underbanked globally.

Completed (2024 - 2025)

Phase 1: Core Infrastructure (Q4 2024)

  • OK Web3 Dashboard - Modern glassmorphism design system
  • OK Authentication System - Email/password with JWT sessions
  • OK KYC/Registration - Compliance and verification workflow
  • OK Database Architecture - PostgreSQL with 25 tables
  • OK API Layer - 105 production endpoints
  • OK Translation System - i18next with Spanish, English, Portuguese
  • OK Role Management System - 8-level hierarchy with auto-assignment

Phase 2: Financial Features (Q1 2025)

  • OK Tanda System - Group creation and management
  • OK Commission System - Coordinator rewards (90/10 split)
  • OK Transaction Engine - Deposits, withdrawals, transfers
  • OK Wallet Integration - Balance history charts, transaction filters
  • OK Admin Panel - User management, KYC review, audit logs

Phase 4: Advanced Features (Q3-Q4 2025)NEW

  • OK Lottery Predictor - ML-powered predictions for Honduras "La Diaria"
    • XGBoost v3 with time-slot specific models

    • Gamification: achievements, points, leaderboards

  • OK Mining System - Daily reward collection
    • Tier progression (Bronze/Silver/Gold)
    • Streak bonuses and achievement points
    • Anti-fraud detection
  • OK WebSocket Live Lottery - Real-time tanda turn assignments
    • Synchronized countdown across all members

    • Multi-position support (N positions per member)
  • OK Notification System - Push notifications for payments, results
  • OK Security Hardening
    • Rate limiting (nginx + fail2ban)
