La Tanda API Documentation

现代化的OAuth 2.0认证和API集成平台

快速开始 (5分钟)

在5分钟内完成La Tanda API集成并开始调用第一个API

# 1. 获取访问令牌 curl -X POST https://my.tanda.co/oauth/token \ -H "Content-Type: application/json" \ -d '{ "grant_type": "password", "client_id": "your_client_id", "client_secret": "your_client_secret", "username": "your_username", "password": "your_password" }' # 2. 调用API curl -X GET https://my.tanda.co/api/v2/users \ -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

API 概览

OAuth 2.0 认证

支持Authorization Code和Password两种认证方式，符合行业标准的安全认证流程

权限系统集成

与Tanda平台权限级别完全同步，确保安全的数据访问控制

环境隔离

测试和生产环境完全隔离，确保开发数据安全

多语言SDK

提供Python、JavaScript、Java等多种语言SDK，简化集成开发

OAuth 2.0 认证流程

1

注册应用

在Tanda平台注册应用，获取Client ID和Client Secret

2

选择认证方式

选择Authorization Code（第三方应用）或Password（自有应用）认证

3

获取访问令牌

通过OAuth 2.0流程获取访问令牌

4

调用API

使用访问令牌调用La Tanda API

API 端点

GET /api/v2/users

获取用户列表，支持分页和过滤

查询参数

参数 类型 必需 说明
page integer 否 页码，默认为1
limit integer 否 每页数量，默认为20，最大100
role string 否 按角色过滤：admin, user, guest

响应示例

{ "data": [ { "id": "user_123", "name": "John Doe", "email": "john@example.com", "role": "admin", "created_at": "2026-03-02T16:00:00Z", "updated_at": "2026-03-02T16:00:00Z" } ], "pagination": { "page": 1, "limit": 20, "total": 1, "total_pages": 1 } }
POST /api/v2/users

创建新用户

请求参数

参数 类型 必需 说明
name string 是 用户姓名
email string 是 用户邮箱地址
role string 否 用户角色，默认为user

请求示例

{ "name": "John Doe", "email": "john@example.com", "role": "admin" }

响应示例

{ "id": "user_123", "name": "John Doe", "email": "john@example.com", "role": "admin", "created_at": "2026-03-02T16:00:00Z", "updated_at": "2026-03-02T16:00:00Z" }
GET /api/v2/users/{user_id}

获取指定用户信息

路径参数

参数 类型 必需 说明
user_id string 是 用户ID

响应示例

{ "id": "user_123", "name": "John Doe", "email": "john@example.com", "role": "admin", "created_at": "2026-03-02T16:00:00Z", "updated_at": "2026-03-02T16:00:00Z" }
PUT /api/v2/users/{user_id}

更新用户信息

路径参数

参数 类型 必需 说明
user_id string 是 用户ID

请求参数

参数 类型 必需 说明
name string 否 用户姓名
email string 否 用户邮箱地址
role string 否 用户角色

请求示例

{ "name": "John Doe Updated", "email": "john.updated@example.com", "role": "user" }

响应示例

{ "id": "user_123", "name": "John Doe Updated", "email": "john.updated@example.com", "role": "user", "created_at": "2026-03-02T16:00:00Z", "updated_at": "2026-03-02T16:30:00Z" }
DELETE /api/v2/users/{user_id}

删除指定用户

路径参数

参数 类型 必需 说明
user_id string 是 用户ID

响应示例

{ "message": "User deleted successfully", "deleted_user_id": "user_123" }

SDK 和工具

Python SDK

功能完整的Python SDK，支持自动令牌管理和错误处理

from tanda_api import TandaAPI client = TandaAPI( client_id='your_client_id', client_secret='your_client_secret' ) users = client.users.list()

JavaScript SDK

现代JavaScript SDK，支持浏览器和Node.js环境

const { TandaAPI } = require('tanda-api'); const client = new TandaAPI({ clientId: 'your_client_id', clientSecret: 'your_client_secret' }); const users = await client.users.list();
