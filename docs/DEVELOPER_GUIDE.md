# Developer Guide - la-tanda-web

## Quick Start

### Prerequisites
- Node.js v18+
- npm or yarn

### Installation
```bash
git clone https://github.com/INDIGOAZUL/la-tanda-web.git
cd la-tanda-web
npm install
npm run dev
```

### Build
```bash
npm run build
```
